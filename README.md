# masDivisores

Realización de la kata "Número con mayor cantidad de divisores" (http://www.solveet.com/exercises/Numero-con-mayor-cantidad-de-divisores/153)

## Requisitos

  * node.js
  * npm
  * commander.js (npm install commander)

## Ejecución

    $ bin/masDivisores tope <numero>
